Kapittel 2: Pythons fire grunnleggende typer

Python kan kan lagre og bruke fire grunnleggende typer. Vi sier grunnleggende fordi andre Python-typer vil utelukkende inneholde disse fire typene. På engelsk brukes begrepene Primitive (Grunnleggende) og Non-Primitive. De fire grunnleggende typene er:


  1. Tekst-streng
  2. Flyttall
  3. Heltall
  4. Bollsk verdi

Tildeling

For å lagre en type må vi tildele den ett navn: <variabel_navn>= <type_verdi>. Python har regler for hva som kan være et variabel-navn. Første tegn må være en bokstav eller understrek: _. Etter dette kan man også bruke tall. Det er ikke lov med mellomrom eller andre tegn enn tall, bokstaver og understrek. I programmering brukes STORE_BOKSTAVER som navn på konstanter (verdier som ikke skal/kan endres) og små_bokstaver for variabler (verdier som kan endres). Dette er utelukkende for å gjøre det lettere å lese en kode. I noen språk vil det ikke være mulig å endre en constant etter den er definert. Dette er ikke tilfellet i Python.

		
a = 5
b = 6
c = a + b
print(c)

Tekst_streng

Vi kan lage en tekst streng på to ulike måter, enten ved å bruke en enslig apostrof: ', på hver side av strengen, eller bruke dobbel-apostrof: ", på hver side av strengen.

		
navn1 = "Egil Dahle"
navn2 = 'Egil Dahle'

For Python er det ingen forskjell mellom de to strengene over.

Heltall og flyttall

Et heltall er ett tall uten desimalverdier. Et flyttall er et desimaltall lagret på en datamaskin. I begge tilfellene vil datamaskinen oversette tallet til ett binært tall, men desimaldelen av ett tall vil ha begrensett med lengde i det binære tallsystemet. Dette kan føre til en avrunding som gir feil tall. Prøv koden under for å se et eksempel på slik avrundingsfeil: 

		
a = 9.4 		#Flyttall
b = 9 			#Heltall
c = .4			#Flyttall
d = a - b - c 	#Flyttall
print(d)

I koden over definerer vi fire variabler. Vi ser også at det kommenteres om det er et heltall eller flyttall. Python overser alt etter # på en linje. Dersom et regnestykke inneholder flyttall og heltall blir resultatet ett flyttall.

Boolsk verdi

En bollsk verdi er en verdi som enten er True eller False

		
a = True 	#En sann verdi
b = False	# En usann verdi
c = 0		# En usann verdi
d = 1		# En sann verdi
d = "Hei"	# En sann verdi
d = ""		# En usann verdi

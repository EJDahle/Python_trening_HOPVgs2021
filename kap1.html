Kapittel 1: Det grunnleggende

Når vi skriver et program i Python lager vi kommandoer som datamaskinen skal utføre. Det enkleste programmet vi kan lage i Python er:

		
print("Hello World!")

Python fungerer veldig rett frem: Programmkode→Python-tolker→Output. For å kjøre et Python-program må vi ha Python installert, for eksempel fra https://www.python.org. Programmet over kan skrives i Notisblokk, og så lenge filen er lagret med .py ending vil Python prøve å kjøre programmet.

Det finnes flere ulike IDE-er som kan kjøre Python-kode. Et IDE (Integrated development environment) lar deg skrive kode og kjøre denne direkte i IDE-en. De gir også skrivehjelp, som for eksempel ord-forslag og farging av ulike ord. Med Python følger IDLE, som kun farger ord, men som ikke gir annen hjelp. Dersom du ikke vil bruke en IDE for å kjøre koden over, men kjøre filen direkte vil du se at den forsvinner med en gang. Du kan derfor be programmet vente på bruker-input. Programmet vil da vente til du tykker enter før det lukkes: 

		
print("Hello World!")
input()

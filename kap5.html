Kapittel 5: If, else og elif

I Python fungerer if veldig likt som en while løkke. Den avgjør om koden inni if-påstanden skal kjøres bassert på om det er en boolsk verdi etter if eller ikke. Vi ser derfor først på de logiske operatorene igjen:

NavnSymbolSyntaks
Er lik===
Ulik≠!=
Større>>
Større eller lik≥>=
Mindre<<
Mindre eller lik≤<=
 

5 == 10  	# False
5 == 5 		# True
20 < 5		# False
a = "Egil"
b = "egil"
a == b		# False
a != b		# True

Vi kan ha flere if-påstander etter hverandre. Python vil da skjekke hver og en etter tur.


a = 5
b = int(input("Oppgi et heltall:"))
if 5 < 10:
	print("5 er mindre enn 10")
if b != a:
	print(f"{a} er ikke lik {b}")

Dersom vi vil at Python skal kjøre en kode dersom en if-påstand ikke stemte kan vi bruke else.


a = int(input("Oppgi et heltall:"))
b = int(input("Oppgi et annet heltall:"))

if b != a:
	print(f"{a} er ikke lik {b}")
else:
	print(f"{a} er lik {b}")

If kan alltid stå alene, mens else må komme etter en if eller en elif. Vi kan også ha flere if og elif etter hverandre, men bare ha en else. elif er ellers en blanding av if og else; dersom påstanden i den forrige if-påstanden eller elif-påstanden ikke stemmte testes denne elif-påstanden. Bare en print-streng vil vises når koden under har kjørt.


a = int(input("Oppgi et heltall:"))
b = int(input("Oppgi et annet heltall:"))

if b != a:
	print(f"{a} er ikke lik {b}")
elif b < a:
	print(f"{a} er større en {b}")
elif b > a:
	print(f"{a} er mindre en {b}")
else:
	print(f"{a} er lik {b}")

Vi kan nå fullføre programmet fra forrige kapittel, der vi skulle finne ut hvem som er høyest i klassen.


antall_elever = int(input("Venligst oppgi antall elever i klassen: "))
elev_navn = []
elev_høyde = []
for i in range(antall_elever):
	elev = input("Oppgi navn: ")
	høyde = int(input(f"Oppgi høyden til {elev} i cm: "))
	elev_navn.append(elev)
	elev_høyde.append(høyde)
	
høyeste_verdi = 0
elev_nummer_høyeste = 0
for elev_nummer in range(len(elev_navn) - 1):
	ny_høyde = elev_høyde[elev_nummer]
	if ny_høyde > høyeste_verdi:
		høyeste_verdi = ny_høyde
		elev_nummer_høyeste = elev_nummer
	elev_nummer = elev_nummer + 1

print(f"{elev_navn[elev_nummer_høyeste]} er den høyeste eleven, med sine {høyeste_verdi}cm")
